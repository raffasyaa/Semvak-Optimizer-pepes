# How To Install

1 . Copy This Bash Script And Paste In Terminal

```
bash <(curl -s https://raw.githubusercontent.com/raffasyaa/Semvak-Optimizer-pepes/main/Tytyd_OP.sh) install
```


## How To Uninstall

1 . Copy This Bash Script And Paste In Terminal


```

```
